NAMA APLIKASI: EcoPoint 2025
DESKRIPSI:
EcoPoint adalah aplikasi pengelolaan sampah pintar berbasis Android yang membantu pengguna mencatat setoran sampah (plastik, kertas, logam, dll), mendapatkan poin (XP), dan naik level (Warga Baru -> Juragan Loak -> Sultan Sampah). Aplikasi ini dilengkapi fitur deteksi jenis sampah otomatis menggunakan AI (Machine Learning).

FITUR UTAMA:
1. Otentikasi (Login & Register).
2. Dashboard Interaktif dengan Gamifikasi (Level & Progress Bar).
3. Input Sampah dengan Deteksi AI (Kamera) atau Manual.
4. Riwayat Transaksi (CRUD: Read, Update, Delete).
5. Statistik Mingguan (Grafik Batang).

CARA MENJALANKAN:
1. Buka project di Android Studio (Ladybug / Jellyfish / Koala).
2. Sync Project dengan Gradle.
3. Jalankan di Emulator atau Device Fisik (Min SDK 24).
4. Lakukan Register akun baru.
5. Login dengan akun yang baru dibuat.

LIBRARY YANG DIGUNAKAN:
- Jetpack Compose (UI)
- Room Database (Penyimpanan Lokal)
- Google ML Kit (Image Labeling untuk AI)
- MPAndroidChart / Custom Canvas (Grafik)
- Coroutines & Flow (Asynchronous)
